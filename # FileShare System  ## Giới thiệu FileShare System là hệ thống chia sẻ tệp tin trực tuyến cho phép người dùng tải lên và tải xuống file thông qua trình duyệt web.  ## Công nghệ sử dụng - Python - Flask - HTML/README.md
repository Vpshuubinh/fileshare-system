# FileShare System

## Giới thiệu
FileShare System là hệ thống chia sẻ tệp tin trực tuyến cho phép người dùng tải lên và tải xuống file thông qua trình duyệt web.

## Công nghệ sử dụng
- Python
- Flask
- HTML/CSS
- Bootstrap

## Chức năng chính
- Upload file
- Download file
- Quản lý danh sách file
- Giao diện web đơn giản

## Cài đặt

1. Clone repository
2. Cài thư viện
pip install -r requirements.txt


3. Chạy chương trình


python app.py


4. Mở trình duyệt


http://127.0.0.1:5000
